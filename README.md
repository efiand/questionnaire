# questionnaire

Тестовое задание в CSSSR.
[Опубликованная версия](https://efiand.github.io/questionnaire)

* `gulpfile.js` – таскраннер (задачи – в `app/tasks`).
* `package.json` – настройки сборки.

## Сборка

* `yarn start` – сборка dev-версии с локальным сервером.
* `yarn build` – сборка в продакшн.
